Components & templates

Every section that builds the Dad's Club site, abstracted from its page context. Each block is defined once as an HTML <template> and rendered here as both a live preview (using the production stylesheet) and its source markup.

Pair with visual-direction.md for tokens, and *-wireframe.md for composition rules per page.

Foundations

Color

Restrained palette. One ink, one paper, one warm accent — used sparingly. No color blocks behind content.

Paper
#F6F2EA · background
Ink
#1B1714 · text
Accent
#A24E2B · hover, <3% of view
Rule
#D9CFBE · dividers, borders
Muted ink
#6B6258 · captions, meta

Type scale

Source Serif 4 + Inter. Serif does the work; sans does the labels. Mobile sizes shown — desktop scales up at ≥1025px (see styles.css).

Atoms

CTA link

Typographic CTA with thin underline and caret. No filled buttons anywhere in the system. Hover shifts to accent + nudges caret 3px right.

Sections


      

        

          
Hero

          
Asymmetric: eyebrow + headline left (cols 1–5), photo right (cols 6–12). Stacks at mobile with photo above headline. No CTA in the hero — nav handles Join.

        

        

          
          

        

        
Markup

      


      

        

          
Indent body block

          
The workhorse layout: heading + body + CTA, indented to cols 2–8. Used anywhere long-form copy lives. Add an eyebrow above only when the section needs a label its heading + body don't already provide (e.g. "Next Gathering", "Past Gatherings") — most body blocks don't.

Next Gathering

          
Single-event block. Heading + hairline + date · time · location + two CTAs. Used on the home page; off-season state swaps the body for a "Join the list" line (see home-wireframe.md).

In Practice grid

          
Asymmetric photo grid: one large (7 cols) + two stacked smaller (4 cols). Editorial choice — resist a tidy 3-up.

        

        

          
          

        

        
Markup

      


      

        

          
Instagram tiles

          
6 most-recent thumbnails. 2-up at S, 3-up at M, 6-up at L. 1px Rule hairline border per tile. Hides itself if the embed fails — never shows broken state.

Newsletter

          
Separate from Join (Google Form). This is the on-domain passive-interest list. Inline at M/L, stacked at S. Replaces with italic "Thanks for submitting!" on submit.

Event list row

          
One row per event (recurring series expand into individual rows). Hairline rules between. Whole row is clickable to the detail page; RSVP link is the explicit primary action.

Past gatherings grid

Event meta row

Sticky RSVP bar

          
Mobile-only on the event detail page. Appears when the inline RSVP scrolls out of view. The one place the design breaks "no chrome" — RSVP is this page's whole purpose.

Contact form

          
First/Last name side-by-side at M+, stacked at S. Email + Message always full-width. Labels above fields (not floating). Hairline borders, no fills.
